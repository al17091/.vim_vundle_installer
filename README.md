# Installer .vimrc for vundle and Plugin

This is Installer for .vimrc and vundle(plugin manager)

# requirement

* git
* ctags

を必要をします。

# Installation

linux 対応
'''in terminal
git clone https://github.com/al17091/vimrc_git.git
cd vimrc_git
sh ./installer.sh
'''

# Note

ctagsとgitがない場合、インストールせずに終了します。
